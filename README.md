## 🌟 我的个人博客

这是一个基于 **Gatsby.js** 构建的极简静态博客，使用 Markdown 编写文章，专注于阅读体验与性能。

* ✍️ 用 Markdown 轻松写作
* 🚀 静态生成，加载飞快
* 📱 响应式设计，适配手机与桌面
* 🔍 内置 SEO 优化，支持 Open Graph 和社交分享卡片

部署于 [Netlify](https://www.netlify.com/)（请根据实际平台修改，如 Vercel 或 GitHub Pages）。

---

### 本地运行

```bash
npm install
npm run develop
```

启动后访问 `http://localhost:8000` 即可预览。
